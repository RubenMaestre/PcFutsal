# FRONTEND — Scripts y Recursos

Esta carpeta contiene scripts y recursos relacionados con el frontend de PC FUTSAL.

## Propósito

Guardar scripts auxiliares, utilidades y recursos del frontend Next.js que no forman parte del código principal de la aplicación.

## Contenido típico

- Scripts de build y optimización
- Utilidades Next.js auxiliares
- Scripts de procesamiento de datos
- Helpers y utilidades
- Scripts de generación de contenido
- Scripts de análisis y métricas
- Scripts de mantenimiento

## Reglas

- Nombrar archivos de forma descriptiva
- Documentar el propósito de cada script
- Registrar en el diario cuando se crean nuevos scripts
- Mantener scripts organizados y documentados
