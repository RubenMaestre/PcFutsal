# BACKEND — Scripts y Recursos

Esta carpeta contiene scripts y recursos relacionados con el backend de PC FUTSAL.

## Propósito

Guardar scripts auxiliares, utilidades y recursos del backend Django que no forman parte del código principal de la aplicación.

## Contenido típico

- Scripts de migración de datos
- Utilidades Django auxiliares
- Management commands auxiliares
- Scripts de procesamiento de datos
- Scripts de importación/exportación
- Utilidades para scraping
- Scripts de mantenimiento de base de datos

## Reglas

- Nombrar archivos de forma descriptiva
- Documentar el propósito de cada script
- Registrar en el diario cuando se crean nuevos scripts
- Mantener scripts organizados y documentados
